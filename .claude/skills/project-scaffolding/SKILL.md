Create a clean, repeatable project foundation.

## When to Use

- Bootstrapping a repo for Claude Code usage

## How It Works
1. Create frontend, backend, docs, and infra directories
2. Initialize package managers
3. Add baseline config files
4. Verify the project runs

## Output
A ready-to-use project scaffold.
