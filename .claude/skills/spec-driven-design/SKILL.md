## Purpose
Prevent ad-hoc design and enforce consistency.

## When to Use
- Designing APIs
- Defining data models
- Coordinating multiple components

## How It Works
1. Identify core entities
2. Select appropriate Spec-Kit Plus specs
3. Define interfaces before implementation

## Output
Clear design specifications.
