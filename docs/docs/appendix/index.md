# Appendices

These appendices provide supplementary information to support your learning throughout the course.

## Available Appendices

### [Appendix A: ML Concepts for Roboticists](/appendix/ml-concepts-roboticists)

Conceptual overview of machine learning fundamentals needed for understanding VLA systems. Covers neural networks, embeddings, and transformers at an intuitive level.

### [Appendix B: Hardware Requirements](/appendix/hardware-requirements)

Detailed hardware specifications for running the course materials, including GPU requirements for Isaac Sim and recommended development configurations.

### [Appendix C: ROS 2 Installation](/appendix/ros2-installation)

Step-by-step guide for installing ROS 2 Humble on Ubuntu 22.04, including common troubleshooting tips.

### [Appendix D: Simulation Installation](/appendix/simulation-installation)

Installation guides for Gazebo Harmonic and NVIDIA Isaac Sim, including GPU driver setup and verification procedures.
